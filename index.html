jPulse.js
jPulse is a jQuery plugin, to add «pulse-like» effect to an element. Tested with jQuery 1.9.1.
Download plugin from the link below and include it in your website: 
<script type="text/javascript" src="jQuery.jPulse.min.js"></script>
